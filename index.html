Le HTML est un langage informatique utilisé sur le web. Ce langage est utilisé pour créer des pages web. L'acronyme sigifie HyperText Markup Language, ce qui signifie en français "langage de balisage hypertexte". Cette signification porte bien son nom puisqu'effectivement ce langage permet de réaliser de l'hypertexte à base d'une structure de balisage.
